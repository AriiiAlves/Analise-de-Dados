# People-Data-Analyzer (pt-br)

Análise de dados de um dicionário contendo as seguintes informações a respeito de pessoas: nome, sexo, idade, cidade, estado e estado civil.

Frameworks usados:
- pandas
- mathplotlib
- fpdf

# People-Data-Analyzer (en)
